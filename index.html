Quick setup

Follow the steps for your device, then continue.

📱 iPhone / iPad

  1. Open this website in Safari browser (this is required — other browsers won't work).
  2. Tap the Share button (⬆️ square with arrow) at the bottom or top-right of the screen.
  3. Select "Add to Home Screen".
  4. Open the app from your Home Screen.
  5. Allow notifications when prompted — this is required to receive alerts.
If notifications don't arrive: Settings → Apps → find the app → Notifications → Enable. If the app doesn't appear in Settings, open it again and repeat the signup — iOS will show the permission popup.

🤖 Android

  1. Open this website in Chrome browser on your Android phone.
  2. Tap the menu icon (⋮) in the top-right corner and select "Add to Home screen" or "Install app".
  3. Confirm the installation and open the app from your Home Screen.
  4. Allow notifications when prompted — this is required to receive alerts.
  5. Important: Don't fully close the app — just minimize it. Notifications only work when Chrome is running in the background.
  6. To prevent the app from being closed automatically: Settings → Apps → Chrome → Battery → Select "Unrestricted"
If notifications still don't arrive: Settings → Apps → Chrome → Notifications → Enable all

💻 Desktop

  1. Open this website on your computer.
  2. Chrome: Click the install icon (💻↓ computer with arrow) in the top-right corner of the address bar, then click "Install".
  3. Safari: Click the Share button (⬆️ square with arrow) in the top-right or bottom of the screen, then select "Add to Dock".
  4. Allow notifications when prompted — this is required to receive alerts.
If notifications don't arrive: Check your system settings. Windows: Settings → System → Notifications → Make sure Chrome/Safari is enabled. Mac: System Preferences → Notifications → Find your browser → Allow notifications.

Free NC DMV appointment notifications (unofficial helper tool). Author: 💼 LinkedIn This free tool automatically monitors the official North Carolina DMV appointment system and sends instant browser notifications when new appointment slots become available.
